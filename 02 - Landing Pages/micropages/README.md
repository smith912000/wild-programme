# WILD Micropages — Index

**Three parallel doors, one room.**

Each micropage is a self-contained initiatory experience built from the same architecture. Different hooks for different demographics, all funnelling into the unified integrated landing page on Branch A. The "ad" you share is one of these micropage URLs — the experience itself is the ad.

---

## The three micropages

| # | File | URL | Persona | Hook |
|---|---|---|---|---|
| 1 | `01_the_hidden_third.md` | `/i/hidden-third` | Universal entry | "You sleep about 8 hours a night. You have never been awake for one of them." |
| 2 | `02_the_fraction.md` | `/i/fraction` | Latent-capability / truth-seeker | "You have been working with a fraction of what is in you. You have been told that fraction is all there is." |
| 3 | `03_the_other_version.md` | `/i/other-version` | Identity-shift / mid-life / contemplative | "Every night you become someone you have never met. They have always been you." |

## Shared specs (defined once in Micropage 1, inherited by 2 and 3)

- **Visual:** pure black background, off-white text (`#F4F1ED`), deep indigo accent (`#2D1B69`), Cormorant Garamond typeface
- **Audio:** ambient drone (60-80Hz) + strike thump (~50Hz) + sustained note (200-300Hz) + recognition-gate chime (~440Hz) + ascending/descending chords for branches
- **Recognition gate:** *"This applies to me."* / *"I am not sure yet."* — Branch A continues, Branch B dead-ends gently with no email capture
- **Tech:** single HTML page with Howler.js for audio, ~4-6 hours dev build per micropage

## How each one is different

| Aspect | Hidden Third | Fraction | Other Version |
|---|---|---|---|
| Persona served | Universal | Truth-seeker | Identity-shift |
| Strike word | "never" | "all" | "always" |
| Unique visual | Eight notches animating below the first line | Partial circle that doesn't complete (until Branch A) | Faint reflection of text behind primary line |
| Branch A reveal | "Then we go to the next door." | "Then you already know what is being asked of you." | "Then it is time you met." |
| Audio extra | None — baseline | None — baseline | Subtle secondary drone at 40Hz for the "two presences" |

## Routing (locked)

**Launch config:** Single micropage → integrated landing page.
- Each URL is a parallel viral unit, all leading to the same destination
- Lower drop-off than chained routing
- Highest signal at the integrated landing

**Future config (save for re-engagement):** Chained micropages.
- Hidden Third → Fraction → Other Version → integrated landing
- Deeper initiation arc for returning visitors
- Add 2-3 months post-launch once first-time funnel is measured

## Anti-fabrication note

All three micropages contain only defensible claims — no stats, no fabricated citations, no community-size claims, no invented success rates. The hooks are philosophical/structural claims about human experience that the manifesto already establishes. The recognition gate routes via reader self-identification, not claimed evidence.

## Brand-health metric to watch

The single most important number once micropages are live: **recognition_no rate** — the percentage of visitors who choose "I am not sure yet" at the gate.

| Range | Diagnosis |
|---|---|
| > 70% | Ad targeting is wrong — drawing people who don't recognise the manifesto's truth |
| 25-50% | Healthy — the gate is filtering meaningfully |
| < 10% | Gate is too lenient — anyone passes — wasted exclusivity signal |

Track per-micropage. The split will tell you which hook resonates with which audience.
